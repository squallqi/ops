# kjyw 快捷运维


## 项目简介
- 快捷运维 kjyw，运维脚本工具库，项目基于shell开发
- 简单 高效 快捷！
- 实现快速安装nginx、mysql、php、redis、nagios运维经常使用的脚本等等... 
- Linux下很多操作命令，都可以通用化，脚本化。
- 脚本化后，可以结合一些自动化工具，批量部署，比如可以用ansible来批量执行脚本，就可以批量部署服务器业务。
- 这里面的脚本是运维经常使用的脚本，方便大家使用！
